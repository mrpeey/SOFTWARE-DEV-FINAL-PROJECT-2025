# EduConnect Lesotho Digital Library Demo Script

## 1. Introduction
- Welcome participants and introduce the project vision: universal access to quality digital education resources for Lesotho.

## 2. Registration & Login
- Show registration page: upload profile image, select user role (student, teacher, librarian, admin).
- Register a new user and log in.
- Display user profile image and role in navbar.

## 3. Book Browsing & Search
- Browse book catalog by category, author, or keyword.
- Demonstrate advanced search and filters.

## 4. Book Reader Features
- Open a book in reader mode.
- Show navigation (next/prev page), font size, theme, and accessibility controls.
- Read first 3 pages online.
- Add notes and bookmarks.
- Show reading progress indicator.

## 5. Offline Access
- Download ebook for offline reading (admin-only for free download).
- Disconnect internet and open offline.html to demonstrate offline access.

## 6. Role-Based Access
- Log in as admin: show dashboard, manage users/books/subscriptions, download/delete/edit books.
- Log in as librarian: view-only admin dashboard, no edit/delete permissions.
- Log in as regular user: restricted access to admin features.

## 7. Mobile & Accessibility
- Resize browser to mobile view: demonstrate responsive design.
- Use keyboard navigation and screen reader features.

## 8. Security & Performance
- Show CSRF protection, secure file uploads, password hashing.
- Demonstrate fast loading and optimized performance.

## 9. SDG Impact
- Explain how the platform supports SDG 4, 9, and 10.
- Highlight inclusivity, innovation, and reduced inequalities.

## 10. Q&A and Closing
- Invite questions and feedback.
- Share contact info and next steps for deployment.

---

*This script ensures a comprehensive, professional demo of all features and SDG impact.*
